Features

  • QR-Validierung & Sicherheitscode
  • Druck- & Einlöseverfolgung
  • Echtzeitprüfung für Apotheken
  • Lizenzmanagement (Stripe, PayPal, Klarna)
  • Patientenakte & Verlaufsdokumentation
  • Digitale Signatur & Audit-Protokolle
  • Videosprechstunde-Modul (Beta)

Installation

git clone https://github.com/at-medical/prescriptcheck.git
cd prescriptcheck
npm install
npm run dev

Nutzung

  • Rezepte digital erstellen und drucken
  • Validierung über Apotheke
  • Statusverfolgung & Statistik
