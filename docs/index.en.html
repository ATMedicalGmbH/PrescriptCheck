Features

  • QR validation & security code
  • Tracking of print & redemption status
  • Real-time pharmacy verification
  • License management (Stripe, PayPal, Klarna)
  • Patient records & medication history
  • Digital signature & audit logs
  • Video consultation module (beta)

Installation

git clone https://github.com/at-medical/prescriptcheck.git
cd prescriptcheck
npm install
npm run dev

Usage

  • Issue & manage prescriptions
  • Verify via pharmacy interface
  • Track status & view analytics
